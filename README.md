# superstore-sales-analysis
My first data analysis project using Python and Jupyter Notebook
## Tools Used
- Python
- Jupyter Notebook
- Pandas, NumPy, Matplotlib, Seaborn

## Key Questions Answered
- Which region generates the most profit?
- How do sales trend over time?
- What is the impact of discounts on profit?

## Dataset
Superstore Sales Dataset from Kaggle
